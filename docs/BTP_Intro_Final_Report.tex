\noindent {\LARGE \textbf{Introduction}}

\vspace{1.5em}

\noindent This report serves as the comprehensive documentation for our BTP-1. As part of BTP-1 we implemented 3 separate tasks as given below:-

\begin{enumerate}[label=\textbf{Task \arabic*:}, leftmargin=2cm]
    \item \textbf{Knowledge Graph Paper Implementations:} For this task each member implemented a separate research paper related to Knowledge Graphs. Each of us then also did a downstream task on top of these implementations. The papers are:-
    \begin{itemize}
        \item Automated Knowledge Graph Construction Pipeline Using Large Language Models (\href{https://arxiv.org/pdf/2403.08345}{link})
        \item Integrating Knowledge Graphs and Vector Retrieval for Enhanced Information Extraction from Financial Documents (\href{https://dl.acm.org/doi/pdf/10.1145/3677052.3698671}{link})
        \item Scaling Up Knowledge Graph Creation to Large and Heterogeneous Data Sources (\href{https://www.sciencedirect.com/science/article/abs/pii/S1570826822000397}{link})
        \item Knowledge Graph Creation from Free Text using Neo4j and Stanza (\href{https://dl.acm.org/doi/10.1145/3716554.3716624}{link})
    \end{itemize}
    \item \textbf{REXEL based Knowledge Graph Extraction Pipeline:} We developed an extraction pipeline leveraging the REXEL framework. This system was designed to transform unstructured textual data into structured knowledge representations, facilitating automated graph construction. Repository Link: \url{https://github.com/bibesgotthevibes/btp}
    \item \textbf{Lay Language Conversion:} The final task focused on application of seq2seq and LLMs for the task of Lay Language Conversion for technical medical discharge summaries. We fine-tuned several seq2seq models including IndicBART, BioBART, BioGPT, SciFive and FlanT5. We also tried using LLMs like Llama, Gemini, Mistral for the same with zero shot and few shot prompting techniques. Repository Link: \url{https://github.com/bibesgotthevibes/btp-Project}
\end{enumerate}
